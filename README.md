# ECG Classification

This project is machine learning algorithm to classify/detect arrhythmia.

I use DNNClassifier in tensorflow.



### Environment

---

1. python 3.5.2
2. tensorflow-gpu 1.10.0
3. intel i7-7700K
4. GeForce GTX 1080 x 2 



### Usage

---

1. Run ***ECG_Classification.ipynb***
2. That's all

### Result

---

- Accuracy :  About 75%
